# Description #

The goal is to create a calendar component which allows to graphically display events.

Features:
```
  * Day, week and month view
  * Easily customizable
  * Multi-language support
```

# Dependencies #

| **Library** | **Version** |
|:------------|:------------|
| Apache Commons Collection | 3.1         |
